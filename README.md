# RushOS

> RushOS is currently under development and is planned to be released soon as posible.

> An Debian-based Linux distribution for beginners, designed to adapt and replace all operating systems.

## Overview

**RushOS** is an Debian-based Linux distribution focused on making Debian accessible to beginners while keeping the power, flexibility, and customization that advanced users expect.

It is built to be a single system that adapts to different workflows, desktop preferences, and user skill levels.

## Desktop Environments

RushOS provides two distinct desktop experiences:

### KDE
- Clean and stable
- Beginner-friendly
- Ideal for daily use and productivity

### GNOME
- Lightweight and fast
- Highly customizable
- Designed for advanced and power users

Users can choose the environment that best fits their workflow.

## Features

- Debian Linux base  
- Beginner-friendly defaults  
- KDE and GNOME editions  
- Focus on usability, performance, and aesthetics  
- High customization without complexity  
- Scales from beginner to advanced usage  

## Target Audience

- New Linux users  
- Users migrating from Windows or macOS  
- Users who want one flexible operating system  

## Philosophy

RushOS is built on a simple idea:

One system.  
Multiple workflows.  
No limitations.

## Project Status

RushOS is under active development.  
Features, configurations, and designs may change over time.
